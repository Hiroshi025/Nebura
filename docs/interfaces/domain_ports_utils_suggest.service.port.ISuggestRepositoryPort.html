    Interface ISuggestRepositoryPort

    Interface for the Suggestion Repository Port.

    Remarks

    This interface defines the contract for managing suggestion entities, including creation, upvoting, downvoting, retrieval, and status updates. Implementations should handle the persistence and retrieval logic for suggestions.
