    Class IPBlocker

    Singleton class to manage IP address blocking and moderation.

    This class provides methods to block, unblock, and check IP addresses, as well as middleware for Express to restrict access from blocked IPs. It also handles automatic unblocking of expired IPs and notification via webhooks.

    Example

    const ipBlocker = IPBlocker.getInstance();
    await ipBlocker.blockIP("192.168.1.1", "adminUser", "Abuse detected");
